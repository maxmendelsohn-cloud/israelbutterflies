# Israel's Butterflies

A small tribute to the beauty of Israel's butterflies — a forest-toned field guide to five of
the most frequently recorded species, where three continents meet.

Static site, no build step. Open `index.html`, or serve the folder.

## Stack
- Plain HTML / CSS / JS — zero frameworks.
- Motion is driven by a hand-built engine in `assets/motion.js` + tokens in `assets/motion.css`
  (flying butterflies, scroll reveals, count-ups, the Latin-name marquee, swaying grass).

## Structure
```
index.html        page + content
styles.css        forest field-guide theme
site.js           butterflies, meadow, pollen, reveals, count-ups, marquee
assets/
  motion.css      motion design tokens + keyframes
  motion.js       motion engine
  *.png           species cutouts (transparent)
  favicon.svg
```

## Content
Species notes adapted from the Israeli Butterfly Monitoring Scheme (BMS-IL) and Butterfly
Conservation Europe. Imagery from Wikimedia Commons (public domain & CC0); the Small White is
illustrated for this guide. More species to come.
